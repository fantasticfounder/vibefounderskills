# Vibe Founder Skills

Business strategy skills for vibecoders. Validate ideas, find customers, position your product, price it, launch it, and grow it.

Built for people who can ship an app in a weekend but don't know how to get their first users.

## What's a Vibecoder?

You build apps with AI tools like Cursor, Replit, Bolt, or Lovable. You can ship fast. But nobody taught you the business side - how to validate demand, find customers, set pricing, or grow beyond your first 10 users. These skills fill that gap.

## How to Install

In Claude Code:

```
claude plugins add fantasticfounder/vibefounderskills
```

Or install individual plugins:

```
claude plugins add fantasticfounder/vibefounderskills/ff-idea-validation
claude plugins add fantasticfounder/vibefounderskills/ff-growth
```

## How to Use

Each plugin has individual skills you can trigger by describing what you need:

- "I need to validate my app idea" - triggers validate-idea
- "Help me find my target audience" - triggers find-target-audience
- "What should I charge?" - triggers pricing-strategy

Or run a full workflow command for a comprehensive strategy session:

```
/validate AI-powered habit tracker for gym beginners
/discover-customers SaaS dashboard for freelancers
/position meeting notes tool for remote teams
/price invoice generator for freelancers
/launch habit tracking app with landing page ready
/grow invoice tool with 200 users
```

## What's Inside

Each skill uses established business frameworks adapted for solo founders:

- **Jobs to Be Done** (Christensen) for customer understanding
- **Positioning** (April Dunford) for market differentiation
- **Blue Ocean Strategy** (Kim & Mauborgne) for competitive analysis
- **Mom Test** (Fitzpatrick) for customer interviews
- **Value Equation** (Hormozi) for offer creation
- **Tactical Empathy** (Chris Voss) for selling
- **Hooked Model** (Nir Eyal) for retention
- **Pretotyping** (Savoia) for cheap validation
- **Crossing the Chasm** (Geoffrey Moore) for market entry

Every skill includes web search triggers for real-time market data, competitor research, and community discovery.
